Instructions

Variables

  • Javaban explicit módon meg kell adni a változó típusát (szám, szöveg, stb.).
  • A változó típusa deklaráláskor rögzített, de az értéke később megváltoztatható.
                
                    int a = 42;      // 'a' egy egész szám (int)
                    String b = "Hello"; // 'b' egy szöveg (String)
                    int[] c = {1, 2, 3}; // 'c' egy tömb (array)
                    Map<String, Object> d = new HashMap<>(); // 'd' egy hashmap (hasonló a Ruby hash-hez)
                    d.put("name", "Alice");
                    d.put("age", 30);

Methods

  • A metódusok definiálásához először meg kell adni a láthatóságát (pl. `public`), majd a visszatérési értékének típusát (pl. `void`, ha nincs visszatérési érték), a metódus nevét, és zárójelben a paraméterlistát (ha vannak).
  • A metódus törzse tartalmazza azokat az utasításokat, amiket végre akarunk hajtani.
  • Az opcionális `return` utasítással adhatunk vissza értéket a metódusból (ha a visszatérési típus nem `void`).
                
                    public static void hi() {
                        System.out.println("Hello World!");
                    }
                    hi(); // => Hello World!
                
            
                
                    public static void hi(String name) {
                        System.out.println("Hello " + name + "!");
                    }
                    hi("Jhony"); // => Hello Jhony!

Object-Oriented Programming

  • Az OOP a Java alapvető programozási paradigmája.
  • A kód "objektumok" köré szerveződik, amelyek adatokat (attribútumok) és viselkedést (metódusok) egyaránt tartalmaznak.
  • A `class` kulcsszóval hozunk létre egy új osztályt, ami az objektumok "sablonja".
  • A konstruktor (az osztály nevével megegyező metódus) létrehozza az objektumot, és beállítja az attribútumait.
  • Az attribútumok az objektum adatait tárolják (pl. `nev`, `kor`).
  • Az objektum metódusai az objektum viselkedését határozzák meg (pl. `koszon`, `mutatkozzBe`).
                
                    class Greeter {
                        private String name;

                        public Greeter(String name) {
                            this.name = name;
                        }

                        public void sayHi() {
                            System.out.println("Hi " + name + "!");
                        }

                        public void sayBye() {
                            System.out.println("Bye " + name + ", come back soon.");
                        }
                    }
                
            
                
                    class Box {
                        private int width;
                        private int height;

                        public Box(int w, int h) {
                            width = w;
                            height = h;
                        }

                        public int getWidth() {
                            return width;
                        }

                        public int getHeight() {
                            return height;
                        }
                    }

                    Box box = new Box(10, 20);
                    int x = box.getWidth();
                    int y = box.getHeight();

                    System.out.println("Width of the box is : " + x);
                    System.out.println("Height of the box is : " + y);
