Hash and array

Map

            
                Map<String, Object> profile = new HashMap<>();
                profile.put("name", "Magnus");
                profile.put("profession", "chess player");
                profile.put("ranking", 1);
                profile.put("grandmaster?", true);
                System.out.println(profile.get("name")); // => Magnus
  • Javaban a `Map` interfész (pl. `HashMap`) kulcs-érték párok gyűjteménye.
  • A `Map` objektumokat a `new HashMap<>()` konstruktorral hozzuk létre.
  • A `put(key, value)` metódussal adunk hozzá elemeket, a `get(key)` metódussal pedig lekérjük az értékeket.

Creating Maps

            
                // Üres Map létrehozása
                Map<String, String> months = new HashMap<>();
            
        
            
                Map<String, Integer> H = new HashMap<>();
                H.put("a", 100);
                H.put("b", 200);
                System.out.println(H.get("a")); // => 100
                System.out.println(H.get("b")); // => 200

Bármilyen Java objektumot használhatsz kulcsként vagy értékként, akár tömböt (`List`) vagy más kollekciót is.

Hash Built-in Methods

A Java `Map` interfész számos beépített metódust kínál.

            
                Map<String, String> months = new HashMap<>();
                months.put("1", "January");
                months.put("2", "February");
                Set<String> keys = months.keySet();
                System.out.println(keys); // => [1, 2] (sorrend nem garantált)

Array

            
                List<String> sharks = new ArrayList<>();
                sharks.add("Hammerhead");
                sharks.add("Great White");
                sharks.add("Tiger");

A `System.out.println()` segítségével kiírhatod a teljes listát.

            
                System.out.println(sharks); // => [Hammerhead, Great White, Tiger]

A `size()` metódussal megtudhatod, hogy hány elem van egy listában:

            
                System.out.println(sharks.size()); // => 3
            
        
            
                System.out.println(sharks.get(0)); // => Hammerhead
                System.out.println(sharks.get(sharks.size() - 1)); // => Tiger

Egy nem létező index elérése `IndexOutOfBoundsException` kivételt dob.

Adding Elements to Lists

Az `add()` metódussal adhatsz hozzá elemeket a lista végéhez:

            
                sharks.add("Whale");
                sharks.add("Bullhead");
                System.out.println(sharks); // => [Hammerhead, Great White, Tiger, Whale, Bullhead]
