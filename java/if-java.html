If Else

If

            
                int pen = 10;
                if (pen < 2) {
                    System.out.println("No pens are left in the shop.");
                }
  • Java-ban az `if` utasítás hasonlóan működik, mint Rubyban.
  • A feltételt zárójelek közé kell tenni `()`.
  • A végrehajtandó kódot kapcsos zárójelek közé kell tenni `{}`.
            
                int a = 22;
                if (a >= 18) {
                    System.out.println("You are eligible for voting");
                } else {
                    System.out.println("You are not eligible for voting");
                }

Unless (No Direct Equivalent in Java)

Java-ban nincs közvetlen megfelelője az `unless`-nek. Használhatjuk helyette a `!` (not) operátort az `if` feltételben:

            
                int x = 1;
                if (!(x >= 2)) { // Az !(x >= 2) ugyanaz, mint az x < 2
                    System.out.println("x is less than 2");
                } else {
                    System.out.println("x is greater than 2");
                }

Switch

            
                class Main {
                    public static void main(String[] args) {
                  
                      int number = 44;
                      String size;
                  
                      // switch statement to check size
                      switch (number) {
                  
                        case 29:
                          size = "Small";
                          break;
                  
                        case 42:
                          size = "Medium";
                          break;
                  
                        // match the value of week
                        case 44:
                          size = "Large";
                          break;
                  
                        case 48:
                          size = "Extra Large";
                          break;
                        
                        default:
                          size = "Unknown";
                          break;
                  
                      }
                      System.out.println("Size: " + size);
                    }
                }
                # => Size: Large

Comparison Operators

Szimbólum Leírás
> A „nagyobb, mint” szimbólum a nagyobb érték feltüntetésére szolgál.
<< A „kisebb, mint” szimbólum a kisebb értéket jelzi.
!= A „nem egyenlő” szimbólum azt mutatja, hogy két érték nem egyenlő.
== Az „egyenlő” szimbólum azt mutatja, hogy két érték egyenlő.
<= A „kisebb, mint egyenlő” szimbólum azt mutatja, hogy az érték kisebb vagy egyenlő.
>= A „nagyobb, mint egyenlő” szimbólum azt mutatja, hogy az érték nagyobb vagy egyenlő.
