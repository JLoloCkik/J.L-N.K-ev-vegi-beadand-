Loops

For Loop

                
                    for (int i = 1; i <= 5; i++) {
                        System.out.println("Message number " + i);
                    }
  • A `for` kulcsszó egy ciklus kezdetét jelzi.
  • `int i = 1` inicializálja a ciklusváltozót.
  • `i <= 5` a ciklusfeltétel, a ciklus addig fut, amíg igaz.
  • `i++` növeli a ciklusváltozót minden iteráció után.

Output

                
                    Message number 1
                    Message number 2
                    Message number 3
                    Message number 4
                    Message number 5

While Loop

                
                    int i = 1;
                    while (i <= 5) {
                        System.out.println("Message number " + i);
                        i++;
                    }
  • A `while` kulcsszó egy ciklust jelez.
  • `i <= 5` a ciklusfeltétel.
  • `i++` növeli a ciklusváltozót minden iteráció után.

Output

                
                    Message number 1
                    Message number 2
                    Message number 3
                    Message number 4
                    Message number 5

Do...While Loop

                
                    int i = 1;
                    do {
                        System.out.println("Message number " + i);
                        i++;
                    } while (i <= 5);
  • A `do...while` ciklus legalább egyszer lefut, majd ellenőrzi a feltételt.
  • `i <= 5` a ciklusfeltétel.
  • `i++` növeli a ciklusváltozót minden iteráció után.

Output

                
                    Message number 1
                    Message number 2
                    Message number 3
                    Message number 4
                    Message number 5
