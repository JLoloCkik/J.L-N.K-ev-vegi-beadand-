Instructions

Variables

  • JavaScriptben a `var`, `let` vagy `const` kulcsszavakkal deklarálhatunk változókat.
  • A változó típusa dinamikus, azaz futásidőben változhat.
                
                    let a = 42;      // 'a' egy szám
                    let b = "Hello"; // 'b' egy szöveg
                    let c = [1, 2, 3]; // 'c' egy tömb (array)
                    let d = { name: "Alice", age: 30 }; // 'd' egy objektum (hasonló a Ruby hash-hez)

Methods

  • JavaScriptben a `function` kulcsszóval definiálunk függvényeket.
  • A függvény törzse tartalmazza azokat az utasításokat, amiket végre akarunk hajtani.
  • Az opcionális `return` utasítással adhatunk vissza értéket a függvényből.
                
                    function hi() {
                        console.log("Hello World!");
                    }
                    hi(); // => Hello World!
                
            
                
                    function hi(name) {
                        console.log(`Hello ${name}!`); // Template literálok használata
                    }
                    hi("Jhony"); // => Hello Jhony!

Object-Oriented Programming

  • JavaScriptben az OOP prototípus-alapú, ami különbözik a Java osztályalapú OOP-jétől.
  • Az objektumok prototípusokból örökölnek tulajdonságokat és metódusokat.
  • A `class` kulcsszó (ES6+) egy kényelmesebb szintaxist biztosít az objektumok létrehozásához.
  • A konstruktor (`constructor`) metódus létrehozza az objektumot, és beállítja az attribútumait.
  • Az attribútumok az objektum adatait tárolják (pl. `nev`, `kor`).
  • Az objektum metódusai az objektum viselkedését határozzák meg (pl. `koszon`, `mutatkozzBe`).
                
                    class Greeter {
                        constructor(name = "World") {
                            this.name = name;
                        }

                        sayHi() {
                            console.log(`Hi ${this.name}!`);
                        }

                        sayBye() {
                            console.log(`Bye ${this.name}, come back soon.`);
                        }
                    }
                
            
                
                    class Box {
                        constructor(w, h) {
                            this.width = w;
                            this.height = h;
                        }

                        printWidth() {
                            return this.width;
                        }

                        printHeight() {
                            return this.height;
                        }
                    }

                    const box = new Box(10, 20);
                    const x = box.printWidth();
                    const y = box.printHeight();

                    console.log("Width of the box is : " + x);
                    console.log("Height of the box is : " + y);
