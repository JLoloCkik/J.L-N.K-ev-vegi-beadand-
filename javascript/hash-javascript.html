Hash and array

Hash

            
                const profile = {
                    name: "Magnus",
                    profession: "chess player",
                    ranking: 1,
                    "grandmaster?": true
                };
                console.log(profile.name); // => Magnus
  • JavaScriptben az objektumok kulcs-érték párok gyűjteményei, hasonlóan a Python dictionary-khoz.
  • Az objektumokat kapcsos zárójelek (`{}`) jelölik.
  • Minden érték egy kulcshoz van rendelve kettőspont (`:`) segítségével.

Creating Objects

            
                // Üres objektum létrehozása
                const months = {};

                // Vagy elemekkel inicializálva
                const months = {1: "January", 2: "February"};
            
        
            
                const H = {a: 100, b: 200};
                console.log(H.a); // => 100
                console.log(H.b); // => 200

Bármilyen JavaScript objektumot vagy primitív értéket használhatsz kulcsként vagy értékként, akár tömböt is.

            
                // Objektum létrehozása tömbből kulcsként (nem túl gyakori)
                const myObject = {[1, "jan"]: "January"}; 
                console.log(myObject); // => {"1,jan": "January"}

Object Built-in Methods

A JavaScript objektumoknak számos beépített metódusa van.

            
                const months = {1: "January", 2: "February"};
                const keys = Object.keys(months);
                console.log(keys); // => ["1", "2"]

Array

            
                const sharks = ["Hammerhead", "Great White", "Tiger"];

A `console.log()` függvénnyel kiírhatod a teljes tömböt.

            
                console.log(sharks); // => ["Hammerhead", "Great White", "Tiger"]

A `length` tulajdonsággal megtudhatod, hogy hány elem van egy tömbben:

            
                console.log(sharks.length); // => 3
            
        
            
                console.log(sharks[0]); // => Hammerhead
                console.log(sharks[sharks.length - 1]); // => Tiger (az utolsó elem)

Egy nem létező index elérése `undefined` értéket ad vissza.

Adding Elements to Arrays

A `push()` metódussal adhatsz hozzá elemeket a tömb végéhez:

            
                sharks.push("Whale");
                sharks.push("Bullhead");
                console.log(sharks); // => ["Hammerhead", "Great White", "Tiger", "Whale", "Bullhead"]
