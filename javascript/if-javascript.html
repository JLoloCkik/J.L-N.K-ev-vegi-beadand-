If Else

If

            
                let pen = 10;
                if (pen < 2) {
                    console.log("No pens are left in the shop.");
                }
  • JavaScriptben az `if` utasítás hasonlóan működik, mint Pythonban. A blokkokat kapcsos zárójelek (`{}`) határolják, és a feltétel után zárójelek (`()`) kellenek.
            
                let a = 22;
                if (a >= 18) {
                    console.log("You are eligible for voting");
                } else {
                    console.log("You are not eligible for voting");
                }

Unless

JavaScriptben nincs közvetlen megfelelője az `unless`-nek. Használhatjuk helyette a `!` (not) operátort az `if` feltételben:

            
                let x = 1;
                if (!(x >= 2)) { 
                    console.log("x is less than 2");
                } else {
                    console.log("x is greater than 2");
                }

Switch

            
                function getSize(number) {
                    switch (number) {
                        case 29:
                            return "Small";
                        case 42:
                            return "Medium";
                        case 44:
                            return "Large";
                        case 48:
                            return "Extra Large";
                        default:
                            return "Unknown";
                    }
                }

                let number = 44;
                let size = getSize(number);
                console.log("Size: " + size); // => Size: Large

A JavaScript `switch` utasítása hasonlóan működik, mint a Python `match-case` szerkezete.

Comparison Operators

Szimbólum Leírás
> A „nagyobb, mint” szimbólum a nagyobb érték feltüntetésére szolgál.
< A „kisebb, mint” szimbólum a kisebb értéket jelzi.
!= A „nem egyenlő” szimbólum azt mutatja, hogy két érték nem egyenlő.
== Az „egyenlő” szimbólum azt mutatja, hogy két érték egyenlő.
<= A „kisebb, mint egyenlő” szimbólum azt mutatja, hogy az érték kisebb vagy egyenlő.
>= A „nagyobb, mint egyenlő” szimbólum azt mutatja, hogy az érték nagyobb vagy egyenlő.
