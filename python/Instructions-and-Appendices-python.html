Instructions

Variables

  • Pythonban nem kell explicit módon megadni a változó típusát (szám, szöveg, stb.).
  • A változó első használata dönti el a típusát, de ez később változhat.
                
                    a = 42      # 'a' egy szám (int)
                    b = "Hello" # 'b' egy szöveg (str)
                    c = [1, 2, 3] # 'c' egy lista (list)
                    d = {"name": "Alice", "age": 30} # 'd' egy szótár (dict)

Methods

  • Pythonban a `def` kulcsszóval definiálunk függvényeket.
  • A függvény törzse tartalmazza azokat az utasításokat, amiket végre akarunk hajtani.
  • Az opcionális `return` utasítással adhatunk vissza értéket a függvényből.
                
                    def hi():
                        print("Hello World!")

                    hi()  # => Hello World!
                
            
                
                    def hi(name):
                        print(f"Hello {name}!")  # f-string használata

                    hi("Jhony")  # => Hello Jhony!

Object-Oriented Programming

  • Python is támogatja az OOP-t, de másképp, mint Java.
  • Minden adat típus objektumnak számít Pythonban.
  • A `class` kulcsszóval hozunk létre egy új osztályt, ami az objektumok "sablonja".
  • Az `__init__` metódus a konstruktor, ami létrehozza az objektumot, és beállítja az attribútumait.
  • Az attribútumok az objektum adatait tárolják (pl. `nev`, `kor`).
  • Az objektum metódusai az objektum viselkedését határozzák meg (pl. `koszon`, `mutatkozz_be`).
                
                    class Greeter:
                        def __init__(self, name="World"):
                            self.name = name

                        def say_hi(self):
                            print(f"Hi {self.name}!")

                        def say_bye(self):
                            print(f"Bye {self.name}, come back soon.")
                
            
                
                    class Box:
                        def __init__(self, w, h):
                            self.width = w
                            self.height = h

                        def print_width(self):
                            return self.width

                        def print_height(self):
                            return self.height

                    box = Box(10, 20)
                    x = box.print_width()
                    y = box.print_height()
                    print(f"Width of the box is : {x}")
                    print(f"Height of the box is : {y}")
