Hash and array

Hash 

            
                profile = {
                    "name": "Magnus",
                    "profession": "chess player",
                    "ranking": 1,
                    "grandmaster?": True  # Pythonban True nagybetűvel kezdődik
                }
                print(profile["name"])  # => Magnus
  • Pythonban a dictionary (szótár) kulcs-érték párok gyűjteménye, hasonlóan a Ruby hash-hez.
  • A dictionary-t kapcsos zárójelek (`{}`) jelölik.
  • Minden érték egy kulcshoz van rendelve kettőspont (`:`) segítségével.

Creating Dictionaries

            
                # Üres dictionary létrehozása
                months = {}

                # Vagy elemekkel inicializálva
                months = {"1": "January", "2": "February"}
            
        
            
                H = {"a": 100, "b": 200}
                print(H["a"])  # => 100
                print(H["b"])  # => 200

Bármilyen Python objektumot használhatsz kulcsként vagy értékként, akár listát is.

            
                # Dictionary létrehozása listából (nem túl gyakori)
                my_dict = dict.fromkeys([[1, "jan"]], "January")
                print(my_dict)  # => {(1, 'jan'): 'January'}

Dictionary Built-in Methods

A Python dictionary-k számos beépített metódussal rendelkeznek.

            
                months = {"1": "January", "2": "February"}
                keys = months.keys()
                print(keys)  # => dict_keys(['1', '2'])

Array

            
                sharks = ["Hammerhead", "Great White", "Tiger"]

A `print()` függvénnyel kiírhatod a teljes listát.

            
                print(sharks)  # => ['Hammerhead', 'Great White', 'Tiger']

A `len()` függvénnyel megtudhatod, hogy hány elem van egy listában:

            
                print(len(sharks))  # => 3
            
        
            
                print(sharks[0])  # => Hammerhead
                print(sharks[-1])  # => Tiger (az utolsó elem)

Egy nem létező index elérése `IndexError` kivételt dob.

Adding Elements to Lists

Az `append()` metódussal adhatsz hozzá elemeket a lista végéhez:

            
                sharks.append("Whale")
                sharks.append("Bullhead")
                print(sharks)  # => ['Hammerhead', 'Great White', 'Tiger', 'Whale', 'Bullhead']
