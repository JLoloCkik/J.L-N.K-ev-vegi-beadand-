If Else

If

            
                pen = 10
                if pen < 2:
                    print("No pens are left in the shop.")
  • Pythonban az `if` utasítás hasonlóan működik, mint Javaban, de nincsenek kapcsos zárójelek, és a blokkokat a behúzással jelöljük.
  • A feltétel után kettőspont (`:` ) szükséges.
            
                a = 22
                if a >= 18:
                    print("You are eligible for voting")
                else:
                    print("You are not eligible for voting")

Unless

Pythonban nincs közvetlen megfelelője az `unless`-nek. Használhatjuk helyette a `not` operátort az `if` feltételben:

            
                x = 1
                if not (x >= 2):  # A Java `!(x >= 2)` helyett `not (x >= 2)`
                    print("x is less than 2")
                else:
                    print("x is greater than 2")

Switch

            
                def get_size(number):
                    match number:
                        case 29:
                            return "Small"
                        case 42:
                            return "Medium"
                        case 44:
                            return "Large"
                        case 48:
                            return "Extra Large"
                        case _:  # Default case (similar to Java's default)
                            return "Unknown"

                number = 44
                size = get_size(number)
                print(f"Size: {size}")  # => Size: Large

A Python 3.10-től kezdve a `match-case` szerkezet hasonló funkcionalitást nyújt, mint a Java `switch` utasítása.

Comparison Operators

Szimbólum Leírás
> A „nagyobb, mint” szimbólum a nagyobb érték feltüntetésére szolgál.
<< A „kisebb, mint” szimbólum a kisebb értéket jelzi.
!= A „nem egyenlő” szimbólum azt mutatja, hogy két érték nem egyenlő.
== Az „egyenlő” szimbólum azt mutatja, hogy két érték egyenlő.
<= A „kisebb, mint egyenlő” szimbólum azt mutatja, hogy az érték kisebb vagy egyenlő.
>= A „nagyobb, mint egyenlő” szimbólum azt mutatja, hogy az érték nagyobb vagy egyenlő.
