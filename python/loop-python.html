Loops

For Loop

                
                    for i in range(1, 6):  # 1-től 5-ig (6 már nem)
                        print(f"Message number {i}")
  • A `for` kulcsszó egy ciklus kezdetét jelzi.
  • `i` a ciklusváltozó.
  • `range(1, 6)` egy számsorozatot generál 1-től 5-ig.

Output

                
                    Message number 1
                    Message number 2
                    Message number 3
                    Message number 4
                    Message number 5

While Loop

                
                    i = 1
                    while i <= 5:
                        print(f"Message number {i}")
                        i += 1  # i = i + 1 rövidítése
  • A `while` kulcsszó egy ciklust jelez.
  • `i <= 5` a ciklusfeltétel.
  • `i += 1` növeli a ciklusváltozót minden iteráció után.

Output

                
                    Message number 1
                    Message number 2
                    Message number 3
                    Message number 4
                    Message number 5

Do...While Loop

Pythonban nincs közvetlen megfelelője a `do...while` ciklusnak. Helyette egy `while True` ciklust használhatunk, és a ciklus belsejében egy `if` feltétellel megszakíthatjuk (`break`):

                
                    i = 1
                    while True:
                        print(f"Message number {i}")
                        i += 1
                        if i == 6:
                            break

Output

                
                    Message number 1
                    Message number 2
                    Message number 3
                    Message number 4
                    Message number 5
