Instructions

Varible

  • Rubyban nem kell explicit módon megadni a változó típusát (szám, szöveg, stb.).
  • A változó első használata dönti el a típusát, de ez később változhat.
        
          a = 42      # 'a' egy szám
          b = "Hello" # 'b' egy szöveg
          c = [1, 2, 3] # 'c' egy tömb
          d = { name: "Alice", age: 30 } # 'd' egy hash

Metódusok

  • A `def` kulcsszóval kezdjük a metódus definícióját, majd következik a metódus neve.
  • Zárójelben adhatjuk meg a metódus paramétereit (ha vannak), vesszővel elválasztva.
  • A metódus törzse tartalmazza azokat az utasításokat, amiket végre akarunk hajtani.
  • Az opcionális `return` kulcsszóval adhatunk vissza értéket a metódusból (ha nem adunk meg, akkor `nil`-t ad vissza).
                    
                    def hi
                      puts "Hello World!"
                    end
                    hi #=> Hello World!
                    
        
                    
                    def hi(name)
                      puts "Hello #{name}!"
                    end
                     hi ("Jhony")#=> Hello Jhony!

Objectiv oriantalt

  • Olyan programozási paradigma, melyben a kód "objektumok" köré szerveződik.
  • Az objektumok adatokat (attribútumok) és viselkedést (metódusok) egyaránt tartalmaznak.
  • Elősegíti a modulárisabb, könnyebben karbantartható kód írását.
  • A `class` kulcsszóval hozunk létre egy új osztályt, ami az objektumok "sablonja".
  • Az `initialize` metódus az osztály konstruktora, ami létrehozza az objektumot, és beállítja az attribútumait.
  • Az attribútumok az objektum adatait tárolják, és `@` jellel kezdődnek (pl. `@nev`, `@kor`).
  • Az objektum metódusai az objektum viselkedését határozzák meg (pl. `koszon`, `mutatkozz_be`).
     
          
            class Greeter
              def initialize(name = "World")
                @name = name
              end
              def say_hi
                puts "Hi #{@name}!"
              end
              def say_bye
                puts "Bye #{@name}, come back soon."
              end
          end

          
        
          
            class Box
               # constructor method
               def initialize(w,h)
                  @width, @height = w, h
               end
            
               # accessor methods
               def printWidth
                  @width
               end
            
               def printHeight
                  @height
               end
            end
            
            box = Box.new(10, 20)
            
            x = box.printWidth()
            y = box.printHeight()
            
            puts "Width of the box is : #{x}"
            puts "Height of the box is : #{y}"
