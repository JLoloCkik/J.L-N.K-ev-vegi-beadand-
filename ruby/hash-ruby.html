Hash and array

Hash

                    
                        profile = {
                          "name" => "Magnus",
                          "profession" => "chess player",
                          "ranking" => 1,
                          "grandmaster?" => true
                        }
                        puts profile["name"] # => Magnus
  • Ruby-ban a hash key-value párok gyűjteménye.
  • A hash-t egy szögletes zárójelekből álló ({}) halmaz jelöli.
  • Minden érték egy kulcshoz van rendelve egy hash rakéta (=>) segítségével.

Creating Hashes

                    
                      months = Hash.new( "month" )

                      or

                      months = Hash.new "month"

                    
        
                    
                        H = Hash["a" => 100, "b" => 200]

                        puts "#{H['a']}" # => 100
                        puts "#{H['b']}" # => 200

Bármilyen Ruby objektumot használhatsz kulcsként vagy értékként, akár tömböt is, így a következő példa is érvényes -

                    
                        [1,"jan"] => "January"

Hash Built-in Methods

A Hash metódus meghívásához szükségünk van egy Hash objektum példányára.

                    
                        Hash[[key =>|, value]* ] or

                        Hash.new [or] Hash.new(obj) [or]
                        Hash.new { |hash, key| block }
                    
        
                    
                        months = Hash.new( "month" )
                        months = {"1" => "January", "2" => "February"}

                        keys = months.keys
                        puts "#{keys}"

Output

                    
                    ["1", "2"]

Array

                    
                        sharks = ["Hammerhead", "Great White", "Tiger"]

A print utasítással kinyomtathat egy teljes tömböt, amely megjeleníti a tömb tartalmát.

                    
                        print sharks # => ["Hammerhead", "Great White", "Tiger"]

A length módszerrel megtudhatjuk, hogy hány elem van egy tömbben:

                    
                        sharks.length # => 3
                    
        
                    
                        puts sharks.first # => Hammerhead
                        puts sharks.last # => Tiger

Egy nem létező index elérésének kísérlete nil értéket ad vissza:

                    
                        sharks[10] # => nill
                    
        
                    
                        sharks[0] # => Hammerhead

Adding Elements to Arrays

Hozzárendelhetünk egy értéket a következő indexhez, ami ebben az esetben a 3 lenne:

                    
                        sharks[3] = "Whale";
                        print sharks # => ["Hammerhead", "Great White", "Tiger", "Whale"]

Ha hozzáadsz egy elemet, és véletlenül kihagysz egy indexet, akkor egy nil elemet hoz létre a tömbben.

                    
                        sharks[5] = "Sand";
                        print sharks # => ["Hammerhead", "Great White", "Tiger", "Whale", nil, "Sand"]

A push módszer helyett a << szintaxist is használhatod, hogy egy elemet hozzáadj egy tömb végéhez:

                    
                        sharks << "Bullhead"
                        print sharks # => ["Hammerhead", "Great White", "Tiger", "Bullhead" ]
