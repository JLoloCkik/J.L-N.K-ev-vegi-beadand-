Loops

For Loop

                    
                        for i in 1..5 do
                            puts "Message number #{i}"
                        end
  • A for kulcsszó egy ciklus kezdetét jelzi.
  • i a mi változónk.
  • Az in kulcsszó jelöli a számok tartományát, amelyen át akarunk lépni.
  • A do kulcsszó jelzi az ismétlendő utasításokat.

Output

                    
                        Message number 1
                        Message number 2
                        Message number 3
                        Message number 4
                        Message number 5

While Loops

                    
                        i = 1
                        while i <   = 5 do
                          puts "Message number #{i}"
                          i = i + 1
                        end
  • A while kulcsszó egy ciklus kezdetét jelzi.
  • i <= 5 azt jelzi, hogy a ciklus addig folytatódik, amíg i kisebb vagy egyenlő 5-nél.
  • A do kulcsszó jelzi az ismétlendő utasításokat.
  • Az end kulcsszó a ciklus végét jelzi.

Output

                    
                        Message number 1
                        Message number 2
                        Message number 3
                        Message number 4
                        Message number 5

Do…While Loops

                    
                        i = 1
        
                        loop do
                            puts "Message number #{i}"
        
                            i = i + 1
                            if i == 6
                                break
                            end
                        end
  • A loop kulcsszó egy ciklus kezdetét jelzi.
  • if i == 6 az a feltétel, amely megszakítja a hurkot.
  • A do kulcsszó jelzi az ismétlendő utasításokat.
  • Az end kulcsszó a ciklus végét jelzi.
  • A break kulcsszó a ciklus végét jelzi.

Output

                    
                        Message number 1
                        Message number 2
                        Message number 3
                        Message number 4
                        Message number 5

Until Loops

                    
                        i = 1
        
                        until i == 6 do
                          puts "Message number #{i}"
                          i = i + 1
                        end
  • A until kulcsszó egy ciklus kezdetét jelzi.
  • Az i == 6 az a feltétel, amely igaz esetben befejezi a hurkot.
  • A do kulcsszó jelzi az ismétlendő utasításokat.
  • Az end kulcsszó a ciklus végét jelzi.

Output

                    
                    Message number 1
                    Message number 2
                    Message number 3
                    Message number 4
                    Message number 5
